# NASDAQ 综合指数 — 深度分析仪表盘

基于 Plotly 的交互式仪表盘，覆盖 NASDAQ 1971–2026 年历史数据（13,941 个交易日）。

**在线预览：** https://max0633.github.io/maxcc-01/

## 截图

用浏览器打开 `dashboard_output.html` 即可查看 — 完全自包含，无需联网。

## 功能

### 重大涨跌事件板块
- 左侧：7 个最大跌幅事件及基本面解读（互联网泡沫、2008 金融危机、COVID 等）
- 右侧：7 个最大涨幅事件及基本面解读（AI 热潮、量化宽松、互联网兴起等）
- 每张事件卡片：分类标签、时间区间、触发因素、基本面、关键事件时间线、教训

### 图表（9 张交互式 Plotly 图表）
| 图表 | 说明 |
|------|------|
| 价格走势 + 成交量 | 对数坐标价格图，含 SMA50/200、布林带、金叉死叉 |
| 历史回撤分析 | 历史回撤幅度，标注 -20% 熊市阈值 |
| 月度收益率热力图 | 年份 × 月份热力图（红黄绿色阶） |
| 牛熊周期 | 市场周期时间轴 |
| 崩盘深度分析 | 横向柱状图展示崩盘幅度 |
| 崩盘对比 | 分组柱状图：回撤幅度 / 下跌天数 / 恢复天数 |
| RSI 指标 | 14 日 RSI，标注超买超卖区间 |
| MACD 指标 | MACD 线、信号线、柱状图 |
| 年度涨跌幅 | 年度收益率柱状图，悬停显示事件原因 |

### UI 设计
- 美式简洁风格 — 纯白背景、system-ui 字体
- 顶部 sticky 导航栏，支持锚点跳转
- 中英双语标题（英文标题 + 中文内容）
- 响应式布局，适配移动端

## 快速开始

```bash
pip install -r requirements.txt
python dashboard.py
```

运行后自动在浏览器中打开 `dashboard_output.html`。

## 项目结构

```
├── dashboard.py            # 主程序 — 生成交互式 HTML 仪表盘
├── crash_analysis.py       # 崩盘自动识别 + 历史事件知识库（7 个事件）
├── cycle_analysis.py       # 牛熊周期识别
├── risk_metrics.py         # 最大回撤、波动率、夏普比率
├── technical_analysis.py   # SMA、RSI、MACD、布林带（仅 2004+ 数据）
├── seasonal_analysis.py    # 月度收益率、热力图数据
├── analysis.py             # 年度收益率控制台分析
├── visualization.py        # matplotlib 静态图表
├── requirements.txt        # pandas, matplotlib, numpy, plotly
├── dashboard_output.html   # 生成的仪表盘（8.3 MB，自包含）
├── index.html              # GitHub Pages 用副本
├── README.md               # English README
└── README_zh.md            # 中文 README
```

## 数据说明

- **来源：** NASDAQ 综合指数日线数据 1971-02-05 至 2026-05-22
- **记录数：** 13,941 个交易日
- **字段：** date, open, high, low, close, volume, amount
- **数据质量：** 1971–2003 仅有收盘价（OHLC 相同，volume=0）；2004+ 完整 OHLCV

## 技术栈

- **Python 3.10+**
- **Plotly** — 交互式图表（JS 内联，完全离线可用）
- **Pandas** — 数据处理
- **NumPy** — 数值计算
- **Matplotlib** — 静态图表（仅 visualization.py 使用）
